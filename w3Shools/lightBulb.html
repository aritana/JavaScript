What Can JavaScript Do?

JavaScript can change HTML attributes values.

In this case JavaScript changes the value of the src(source) attribue of an image.

Explicação:
  • Utilize a tag de imagem, um objeto HTML, nomeando-a com um id. 
                            
                           <img id="myImage" src="pic_bulboff.gif" style="width:100px">
  • Utilize dois botões, alterando a propriedade 'src' do objeto HTML 'imgBulb', criada no passso anterior. Que nada mais é vincular ao botão Turn on a imagem bulboff.gif e ao botão Turn off a imagem bulbon.gif. 
                            
                           <button onclick="document.getElementById('myImage').src='pic_bulbon.gif'">Turn on the light </button>
                        
                        
                           <button onclick="document.getElementById('myImage').src='pic_bulboff.gif'">Turn on the light </button>
